## *Crenicichla* genomics

This repo includes nextflow scripts used for genome assembly, quality assessment, annotation and phylogenetic analysis of *Crenicichla* genomes.

The scripts used for artice "A T2T genome of the lobed-lip Iguazu joana (Crenicichla tuca) reveals widespreadintrachromosomal rearrangements in cichlid fishes" are archived in [CtucaBranch](https://github.com/mylena-s/assembly_project/tree/CtucaBranch)
